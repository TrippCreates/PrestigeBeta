Prestige Project

Project Structure Documentation

Project Overview

The Prestige project is an iOS/macOS application built using Xcode. The project follows a standard Xcode project structure with separate directories for the main application, unit tests, and UI tests.

Project Structure

Main Application

  • Prestige/ - Contains the main application source code

Testing

  • PrestigeTests/ - Contains unit tests for the application
  • PrestigeUITests/ - Contains UI tests for the application

Project Configuration

  • Prestige.xcodeproj/ - Xcode project configuration files

Getting Started

To work with this project:

  • Open the project using Xcode by double-clicking the Prestige.xcodeproj file
  • The main application code is located in the Prestige directory
  • Tests can be found in the PrestigeTests and PrestigeUITests directories
